# Kreativa Studio Arq — Landing Page

Landing page moderna y responsive para **Kreativa Studio Arq**, un estudio de arquitectura, interiorismo y ejecución de proyectos.

El sitio fue desarrollado en **React + Vite**, utilizando información estática e imágenes locales para lograr una experiencia rápida, visualmente atractiva y fácil de mantener.

## Características

* Diseño premium con estética editorial de arquitectura.
* Sitio 100% estático, sin backend ni base de datos.
* Secciones de presentación, servicios, proyectos, interiorismo, planimetría y contacto.
* Imágenes locales optimizadas.
* Diseño responsive para mobile, tablet y desktop.
* Integración de redes sociales y botones de contacto.
* Preparado para deploy en Vercel.

## Tecnologías utilizadas

* React
* Vite
* TypeScript
* CSS
* Assets locales

## Objetivo

Presentar los servicios, proyectos e identidad visual de Kreativa Studio Arq mediante una landing page profesional, elegante y rápida, pensada para captar consultas y reforzar la presencia digital del estudio.
